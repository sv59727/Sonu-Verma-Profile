# Dream Progressive Ventures (DPV) Website

Welcome to the source code for the Dream Progressive Ventures website.

## Project Structure
- `index.html` - The main structure of the website.
- `styles.css` - All styling, colors, and layout rules.
- `script.js` - Interactive functionality (mobile menu, smooth scrolling).

## how to Run
Since this is a static website, you can simply open the `index.html` file in any modern web browser to view the site.

1. Navigate to `e:\Projects\Sonu Verma Profile` in File Explorer.
2. Double-click `index.html`.

## Customization
- **Images**: You can replace the background images in `styles.css` or the `img` tags in `index.html`.
- **Text**: Edit `index.html` to update services, contact info, or bio.
- **Colors**: Adjust `styles.css` variables at the top of the file to change the color scheme.
